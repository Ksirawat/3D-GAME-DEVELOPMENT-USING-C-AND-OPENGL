Simple 3D Game Demo

This is a simple 3D game demo built with C++ and OpenGL. The player can run around a small scene and collect items.

Controls

W / S: Move Forward / Backward

A / D: Rotate Left / Right

Mouse Scroll: Zoom In / Out

ESC: Quit Game

Objective

The goal is to walk around the map and collect all the backpacks.

Credits

This project uses 3D models from Kenney.nl (Mini Market pack).

Built on top of the learnopengl.com framework.
